# Prototype ARTAG — Base stabilisée

Ce dépôt initialise la **reconstruction par blocs** du prototype React ARTAG.

## Itération courante

Cette itération couvre uniquement l'étape **1. Architecture générale** :
- structure d'entrées de l'outil ;
- zones applicatives séparées ;
- cartographie des parcours visuels validés ;
- séparation explicite entre dossier partagé, espace professionnel réservé et pilotage direction.

Aucune logique métier détaillée, aucun calcul d'indicateur et aucun habillage visuel définitif n'est figé à ce stade.
